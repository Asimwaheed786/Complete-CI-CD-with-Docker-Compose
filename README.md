# Docker in CI/CD Pipeline

## Objective
This project demonstrates how Docker is integrated into a CI/CD pipeline to ensure consistent and reliable deployments across environments.

### Learning Outcomes
- Understand the flow of a CI/CD pipeline.
- Use Docker and Docker Compose for building and testing.
- Apply containerization for consistent environments.
- Automate the build-test-deploy cycle.

---

## Tasks

### **Task 1: Define a CI/CD Pipeline Scenario**

#### Objective
Set up a basic pipeline: **Code → Build → Test → Deploy**

#### Steps
1. Push code updates to a GitHub repository.
2. Configure a CI/CD tool (e.g., Jenkins).
3. Build a Docker image from the source code.
4. Run tests inside the Docker container.
5. Deploy the application upon successful test completion.
